# Ducksound
Ducksound é um webapp feito em HTML, CSS e Javascript utilizando o Framework Bootstrap. Ele pesquisa músicas no Soundcloud e disponibiliza elas para Download, isso tudo é feito na parte do cliente e não precisa de um Servidor Web, você pode hospedar este site no Github. Ele irá baixar até as músicas do Soundcloud que não estão disponíveis para Download.

As músicas que estão disponíveis para download, serão baixadas direto do servidor do Soundcloud, mas as que não estão disponíveis serão baixadas do servidor 188.138.17.231 que é mantido pelo Krafta.
